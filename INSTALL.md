# 🚀 설치 및 시작 (Installation)

이 시스템은 작업 대상(Data)과 운영 체제(OS)를 명확히 분리하는 철학을 가집니다. 따라서 이 저장소는 작업 디렉토리(`~/my/github`) 내부가 아닌, **OS 레이어(`~`)에 직접 배치**하는 것을 권장합니다.

## 1. 저장소 클론 (Clone)
기존 작업 영역 밖인 최상위 OS Layer에 소스코드를 클론합니다.

[https://github.com/aimldl/my-os](https://github.com/aimldl/my-os)

```bash
git clone https://github.com/aimldl/my-os ~/my-os
```
