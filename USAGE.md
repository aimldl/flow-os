# 💻 Flow OS Usage
Flow OS는 "시작 → 유지 → 복귀 → 종료"를 하나의 흐름으로 만든다.

## 🚀 전체 Workflow
터미널 → Dashboard → go → 작업 → memo → organize → quit

### 🚀 현재 구조
go      → 실행 진입 (RESUME → NEXT)

memo    → 생각 배출
organize → 정리
next    → 의도 설정
resume  → 상태 저장
quit    → 종료
