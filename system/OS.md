## 구조

~/my

## 사용법
1. 시작: 터미널 → TODO / RESUME 확인 → 바로 실행  
2. 작업: 헷갈리면 RESUME 확인 → 다음 것 실행  
3. 종료: q 또는 qq → gpush + RESUME 업데이트  

## 흐름
tmp → playground → core / projects
